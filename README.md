# Vibe Films

A sleek, minimalist movie recommender web app that suggests films based on your current mood or vibe.

> **Note:** This is a proof of concept with limited film selection. Movie posters are sourced from TMDB, and availability varies by category.

## Features

- **Emotion-based recommendations**: Type how you're feeling and get curated film suggestions
- **Multiple vibe support**: Combine vibes like "happy and nostalgic" or "sad but cozy" for nuanced recommendations
- **9 distinct vibes**: happy, sad, anxious, melancholic, energetic, nostalgic, contemplative, weird, cozy
- **Quick vibe selection**: Click suggestion chips for instant recommendations
- **Curated films with verified posters**: Only displays movies with working poster images
- **Movie details**: Click any film to see year, director, and description
- **Smooth animations**: Elegant fade-in effects and transitions
- **Regenerate options**: Get fresh suggestions for the same vibe(s)
- **Responsive design**: Works beautifully on all screen sizes

## How to Use

1. Open `index.html` in your web browser
2. Type your current emotion/mood in the input field, or click a suggestion chip
3. Browse the 4 recommended films
4. Click any poster to see more details about the film
5. Click "regenerate" to get different recommendations for the same vibe

## Tech Stack

- Pure HTML, CSS, and JavaScript
- No external dependencies or frameworks
- Movie posters from TMDB API
- Responsive grid layout
- Async poster validation to ensure working images

## Current Limitations

- **Limited film library**: Some vibe categories have fewer movies than others
- **Poster availability**: Only movies with verified working poster URLs are displayed
- **No search**: Currently focused on vibe-based browsing only
